Soundly. Solid sound manipulation for your web app.

Sounds are good, you should use them. Whether you are making a game, an interactive experience, or a web-based spreadsheet, there are numerous reasons to tickle your users' eardrums with delicious sonic droplets of bing-bong. What the hell does that mean? Doesn't matter, the good news is that Soundly can help you do it.

Install soundly

For browserify/webpack builds:

npm install soundlyjs

For bower builds:

bower install soundlyjs

Soundly is an ES6 module, so you can just

import soundly from 'soundlyjs'

and then refer directly to Soundly's handy factories:

let appMixer = new soundly.Mixer();
let whoopieCushion = new soundly.Sound('http://mydomain.com/assets/whoopieCushion.mp3');
let appPlayer = new soundly.Player();
appPlayer.outputTo(appMixer.channels[0]);
appPlayer.playSound(whoopieChusion);
